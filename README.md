# flightradar
Game of flight radar
Utilisation
	1.	Entrez les prénoms des deux joueurs.
	2.	Autorisez la géolocalisation (ou cherchez votre ville, ou entrez des
coordonnées à la main).
	3.	Le radar affiche les avions détectés autour de vous : distance, direction,
altitude. Repérez celui que vous voyez réellement dans le ciel et
sélectionnez-le (ou saisissez son indicatif si le radar ne le montre pas).
	4.	Chaque joueur écrit son pronostic (pays d’origine / destination, et en
bonus les villes).
	5.	Cliquez sur “Révéler le vol” : le vrai trajet s’affiche. Les pays corrects
sont cochés automatiquement (vous pouvez corriger à la main en cas
d’orthographe ou d’alias non reconnu) ; validez ensemble les villes.
	6.	Le score s’additionne au fil des manches et reste enregistré dans le
navigateur.
Limites à connaître
	•	Certains avions n’ont pas de route connue dans la base (vols privés,
militaires, jets d’affaires) : passez simplement à un autre avion.
	•	La détection radar dépend de la couverture ADS-B communautaire — dans des
zones très rurales ou certains pays, moins d’avions remontent.
	•	Les scores sont stockés localement dans le navigateur (pas de compte, pas de
synchronisation entre appareils).
